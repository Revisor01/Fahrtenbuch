Neue Datei `frontend/src/components/AddressAutocomplete.js` erstellen. Self-contained Komponente mit:

**Props:**
- `value` (string) — aktueller Eingabewert
- `onChange` (function) — Callback mit neuem Wert (string), wird bei Tippen UND Auswahl aufgerufen
- `placeholder` (string, optional, default "Adresse eingeben")
- `className` (string, optional) — zusaetzliche CSS-Klassen fuer das Input
- `required` (boolean, optional)
- `name` (string, optional) — fuer Formulare

**Verhalten (per D-01, D-02, D-03, D-04, D-08):**
1. Input-Feld rendert mit `className="form-input"` plus uebergebene className
2. Bei Eingabe: `onChange(inputValue)` sofort aufrufen (kontrollierte Komponente)
3. Debounce mit `setTimeout` / `clearTimeout` (300ms) — KEIN lodash (per deep_work_rules)
4. Erst ab 3 Zeichen Nominatim aufrufen (per D-08)
5. Nominatim URL: `https://nominatim.openstreetmap.org/search?format=json&countrycodes=de&limit=5&q=${encodeURIComponent(query)}` (per D-01, D-03)
6. fetch mit Header `Accept: application/json` — kein axios (externer API-Call direkt vom Browser)
7. Ergebnis als Dropdown unter dem Input anzeigen

**Dropdown-Styling (Dark-Mode-kompatibel mit bestehenden Klassen):**
- Container: `absolute z-50 w-full mt-1 rounded-lg shadow-lg border` mit `bg-white dark:bg-gray-800 border-gray-200 dark:border-gray-700`
- Jeder Eintrag: `px-3 py-2 cursor-pointer text-sm` mit `text-value` Klasse
- Hover: `hover:bg-gray-100 dark:hover:bg-gray-700`
- `display_name` aus Nominatim-Antwort anzeigen

**Interaktion (per D-05, D-06):**
- Klick auf Vorschlag: `onChange(suggestion.display_name)` aufrufen, Dropdown schliessen
- Escape-Taste: Dropdown schliessen
- Click-outside: Dropdown schliessen (useRef + useEffect mit document.addEventListener mousedown)
- Bei leerem Input oder weniger als 3 Zeichen: Dropdown schliessen, kein Fetch

**State:**
- `suggestions` (array) — Nominatim-Ergebnisse
- `isOpen` (boolean) — Dropdown sichtbar
- `debounceTimer` via useRef

**Cleanup:** clearTimeout im useEffect cleanup und bei unmount.

Export: `export default AddressAutocomplete;`
