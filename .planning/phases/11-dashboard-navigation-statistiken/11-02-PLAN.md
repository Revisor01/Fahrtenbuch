<!-- monthlyData Struktur aus AppContext -->

monthlyData ist ein Array von Objekten:
```js
{
  yearMonth: "2026-03",           // Format YYYY-MM
  monthName: "March",             // Locale month name
  year: 2026,                     // Number
  monatNr: 3,                     // Number 1-12
  erstattungen: {                 // Object: Traeger-Name -> Betrag
    "Kirchenkreis": 45.60,
    "Gemeinde": 12.30
  },
  abrechnungsStatus: {},
  totalErstattung: 57.90          // Number: Gesamtbetrag
}
```

Nur Monate mit Erstattungen > 0 sind enthalten (gefiltert in fetchMonthlyData).
Sortiert nach Datum absteigend (neueste zuerst).
Enthalt 24 Monate rueckwaerts + 3 Zukunft.

fahrten Array enthaelt Fahrt-Objekte mit `datum` (YYYY-MM-DD String) und `kilometer` (Number).
</interfaces>

   - 12 Balken nebeneinander (flex, gap-1)
   - Jeder Balken: `<div>` mit dynamischer Hoehe relativ zum Maximum. Mindesthoehe 2px auch bei 0 km.
   - Max-Hoehe Container: 160px (h-40)
   - Balken-Breite: `flex-1` (gleichmaessig verteilt)
   - Balken-Farbe: `bg-primary-500` (bestehendes Farbschema)
