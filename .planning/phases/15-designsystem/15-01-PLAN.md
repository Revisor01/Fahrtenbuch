1. **themes.css — Dark-Mode-Hintergrund-Variablen ergaenzen:**
   In JEDEM Theme-Block (classic, sunset, mint, slate, monochrome, teal, vibrant, deep-purple, forest) folgende Variablen NACH den bestehenden secondary-Variablen ergaenzen:
   ```css
   --bg-surface: #ffffff;
   --bg-card: #ffffff;
   --bg-card-highlight: [theme-primary-25 Wert];
   --border-card: [theme-primary-100 Wert];
   ```
   Die highlight/border-Werte je Theme aus den bestehenden primary-25 und primary-100 Werten ableiten.

   Zusaetzlich einen `:root.dark`-Block (oder `html.dark`-Block) fuer Dark-Mode-Overrides ergaenzen, der fuer ALLE Themes gilt:
   ```css
   html.dark {
     --bg-surface: #111827;    /* gray-900 */
     --bg-card: #1f2937;       /* gray-800 */
     --bg-card-highlight: #1e293b; /* slate-800 */
     --border-card: rgba(255, 255, 255, 0.1);
   }
   ```

2. **tailwind.config.js — Erweitern:**
   Im `theme.extend`-Objekt folgende Ergaenzungen:
   ```js
   spacing: {
     'card': '1.5rem',      // 24px — Standard Card Padding
     'card-sm': '1rem',     // 16px — Kompaktes Card Padding
     'card-lg': '2rem',     // 32px — Grosszuegiges Card Padding
     'section': '1.5rem',   // 24px — Abstand zwischen Sektionen
   },
   borderRadius: {
     'card': '0.75rem',     // 12px — Standard Card Radius (zwischen rounded-lg und rounded-xl)
     'card-lg': '1rem',     // 16px — Groesserer Card Radius
   },
   boxShadow: {
     'card': '0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px -1px rgba(0, 0, 0, 0.1)',
     'card-hover': '0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -2px rgba(0, 0, 0, 0.1)',
     'card-elevated': '0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -4px rgba(0, 0, 0, 0.1)',
   },
   backgroundColor: {
     surface: 'var(--bg-surface)',
     card: 'var(--bg-card)',
     'card-highlight': 'var(--bg-card-highlight)',
   },
   borderColor: {
     card: 'var(--border-card)',
   },
   ```

<task type="auto">
  <name>Task 2: Card-Klassen und Dark-Mode-Kontraste in index.css modernisieren</name>
  <files>frontend/src/index.css</files>
  <read_first>frontend/src/index.css, frontend/src/themes.css</read_first>
  <action>
1. **base-container modernisieren:**
   Aktuellen base-container:
   ```css
   .base-container {
     @apply bg-white dark:bg-gray-800
         rounded-lg shadow-sm
         border border-primary-100 dark:border-primary-700
         overflow-hidden
         transition-colors duration-200;
   }
   ```
   Ersetzen durch:
   ```css
   .base-container {
     @apply bg-card
         rounded-card shadow-card
         border border-card
         overflow-hidden
         transition-all duration-200;
   }
   ```
   Dies nutzt die neuen CSS-Variablen aus themes.css und tailwind.config.js.

2. **card-container-highlight modernisieren:**
   Von:
   ```css
   .card-container-highlight {
     @apply base-container p-6
            bg-primary-25 dark:border-primary-900;
   }
   ```
   Zu:
   ```css
   .card-container-highlight {
     @apply base-container p-6
            bg-card-highlight;
   }
   ```

3. **mobile-card modernisieren:**
   Von:
   ```css
   .mobile-card {
     @apply bg-white dark:bg-gray-800
         rounded-lg
         border border-primary-100 dark:border-primary-800
         p-4 space-y-4
         shadow-sm
         transition-all duration-200
         hover:border-primary-200 dark:hover:border-primary-700
         hover:shadow-md;
   }
   ```
   Zu:
   ```css
   .mobile-card {
     @apply bg-card
         rounded-card
         border border-card
         p-4 space-y-4
         shadow-card
         transition-all duration-200
         hover:shadow-card-hover;
   }
   ```

4. **Dark-Mode-Kontraste verbessern — text-label:**
   Von:
   ```css
   .text-label {
     @apply text-primary-600 dark:text-primary-100;
   }
   ```
   Zu:
   ```css
   .text-label {
     @apply text-primary-600 dark:text-primary-300;
   }
   ```
   (primary-300 statt primary-100 gibt mehr Kontrast gegen dark:bg-gray-800)

5. **Buttons modernisieren — groesserer Radius:**
   btn-primary und btn-secondary: `rounded` ersetzen durch `rounded-lg`.

6. **form-input modernisieren:**
   `rounded` ersetzen durch `rounded-lg`, `shadow-sm` ersetzen durch `shadow-card`.

7. **Neue Utility-Klasse fuer Card-Hover ergaenzen** (am Ende des @layer components Blocks):
   ```css
   .card-interactive {
     @apply base-container p-6
            cursor-pointer
            hover:shadow-card-hover
            hover:scale-[1.01]
            transition-all duration-200;
   }
   ```
  </action>
