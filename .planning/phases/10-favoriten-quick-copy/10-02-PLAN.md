2. Nochmal-Handler:
```javascript
const handleNochmal = async (fahrt) => {
  try {
    await axios.post('/api/fahrten', {
      vonOrtId: fahrt.von_ort_id,
      nachOrtId: fahrt.nach_ort_id,
      datum: new Date().toISOString().slice(0, 10),
      anlass: fahrt.anlass,
      abrechnung: fahrt.abrechnung,
      einmaligerVonOrt: fahrt.einmaliger_von_ort,
      einmaligerNachOrt: fahrt.einmaliger_nach_ort,
      kilometer: fahrt.kilometer
    });
    showNotification('Fahrt erstellt', `Fahrt ${fahrt.von_ort_name} → ${fahrt.nach_ort_name} wurde fuer heute eingetragen.`);
    refreshAllData();
  } catch (error) {
    showNotification('Fehler', 'Fahrt konnte nicht erstellt werden.');
  }
};
```

3. Visuell die letzten 3 Fahrten von den restlichen abheben: Dezenter Hinweis "Letzte Fahrten — schnell wiederholen" als kleine Ueberschrift oder Badge, NICHT aufdringlich. Der Nochmal-Button ist nur bei den oberen 3 Eintraegen sichtbar.
