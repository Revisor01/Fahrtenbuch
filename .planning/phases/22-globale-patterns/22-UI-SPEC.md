| Token | Value | Usage |

| sm | 8px | Compact element spacing, gap-2 |

## Component Patterns Contract

### GP-01: Button-Vereinheitlichung

**Ist-Zustand (Problem):**
- btn-primary: h-8 (32px)
- btn-secondary: h-10 (40px)
- btn-destructive: h-8 (32px)

**Soll-Zustand:**
```css
.btn-primary    { height: 40px (h-10); }
.btn-secondary  { height: 40px (h-10); }
.btn-destructive { height: 40px (h-10); }
```

Alle drei Klassen: h-10, px-4, text-sm, rounded-card, whitespace-nowrap.
Alle Inline-Overrides (h-8, h-10 in Komponenten) entfernen.

### GP-02: Card-Background Design-Tokens

**Ist-Zustand (Problem):**
- Dashboard KPI-Cards: hardcoded `bg-emerald-50`, `bg-blue-50`, `bg-purple-50`
- MonthlyOverview: Inline-Style mit `backgroundColor: farbe + "14"`

**Soll-Zustand:**
- KPI-Cards nutzen CSS-Klassen (.kpi-card + .kpi-card-emerald etc.)
- Dynamische Farben per CSS-Variable oder color-mix()
- Kein hardcoded bg-{color}-50 in JSX-Komponenten

### GP-03: KPI-Card CSS-Pattern

Neue CSS-Klasse `.kpi-card` in index.css:

```
.kpi-card {
  rounded-card p-4 shadow-card border border-card
  /* Kein Background — wird per Variante gesetzt */
}
```

Struktur (HTML-Vertrag):
```html
<div class="kpi-card kpi-card-emerald">
  <div class="flex items-center justify-between gap-2">
    <div class="min-w-0">
      <p class="text-xs text-muted mb-1">{Label}</p>
      <p class="text-xl font-semibold text-value truncate">{Wert}</p>
      <!-- Optional: Unterzeile -->
      <p class="text-xs text-muted">{Zusatzinfo}</p>
    </div>
    <{Icon} size={22} class="{icon-farbe} shrink-0" />
  </div>
</div>
```

Icon-Position: rechts (bestehender Dashboard-Standard).
Icon-Groesse: 22px (bestehender Standard, size={22}).

### GP-04: Section-Header CSS-Pattern

Neue CSS-Klasse `.section-header` in index.css:

```
.section-header {
  flex items-center gap-2 mb-3
}
```

Struktur (HTML-Vertrag):
```html
<div class="section-header">
  <{Icon} size={18} class="{icon-farbe}" />
  <h2 class="text-base font-medium text-value">{Titel}</h2>
  <!-- Optional: Anzahl-Badge -->
  <span class="text-sm text-muted">({count})</span>
</div>
```

Icon-Groesse: 18px (einheitlich, ersetzt aktuelle 18/20/22px Mischung).
Heading-Level: h2.
Margin-Bottom: mb-3 (12px) — Dashboard-Standard.
Optionaler Count: text-sm text-muted in Klammern, rechts vom Titel.
