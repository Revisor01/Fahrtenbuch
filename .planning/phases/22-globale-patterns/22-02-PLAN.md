**MonthlyOverview.js:**

1. Zeile 11-13: Die `getCardBg` Funktion ersetzen. Statt `backgroundColor: hexColor + "14"` (Hex-Opacity-Hack) eine saubere Loesung mit hex-to-rgba Konvertierung verwenden:
   ```js
   const getCardBg = (hexColor) => {
     const hex = (hexColor || DEFAULT_FARBE).replace('#', '');
     const r = parseInt(hex.substring(0, 2), 16);
     const g = parseInt(hex.substring(2, 4), 16);
     const b = parseInt(hex.substring(4, 6), 16);
     return { backgroundColor: `rgba(${r}, ${g}, ${b}, 0.08)` };
   };
   ```
   Dies erzeugt ~8% Opacity (gleicher visueller Effekt wie "14" hex = 20/255 = ~8%).
